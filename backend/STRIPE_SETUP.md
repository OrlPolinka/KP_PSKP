# Установка и настройка Stripe для платежной системы

## 📦 Установка зависимостей

```bash
# Перейдите в бэкенд директорию
cd backend

# Установите Stripe
npm install stripe

# Установите Stripe React библиотеку (для фронтенда)
cd ../frontend
npm install @stripe/stripe-js @stripe/react-stripe-js
```

## 🔑 Получение API ключей Stripe

1. Перейдите на [Stripe Dashboard](https://dashboard.stripe.com/)
2. Создайте аккаунт или войдите
3. Перейдите в раздел **Developers** → **API keys**
4. Скопируйте **Test publishable key** (начинается с `pk_test_`)
5. Скопируйте **Test secret key** (начинается с `sk_test_`)

## ⚙️ Настройка переменных окружения

### Бэкенд (.env):
```env
# Stripe тестовые ключи
STRIPE_SECRET_KEY=sk_test_ВАШ_СЕКРЕТНЫЙ_КЛЮЧ
STRIPE_PUBLISHABLE_KEY=pk_test_ВАШ_ПАБЛИЧНЫЙ_КЛЮЧ
STRIPE_WEBHOOK_SECRET=whsec_ВАШ_WEBHOOK_СЕКРЕТ

# URL фронтенда (для редиректов)
FRONTEND_URL=http://localhost:3000
```

### Фронтенд (.env):
```env
REACT_APP_STRIPE_PUBLISHABLE_KEY=pk_test_ВАШ_ПАБЛИЧНЫЙ_КЛЮЧ
```

## 🌐 Настройка вебхуков

### Для локального тестирования:

Используйте **Stripe CLI** для локального туннелирования:

```bash
# Установите Stripe CLI
npm install -g stripe

# Войдите в Stripe
stripe login

# Запустите локальный вебхук
stripe listen --forward-to localhost:8000/api/payments/webhook
```

### Для продакшена:

1. Разверните приложение на хостинге
2. В Stripe Dashboard добавьте URL вебхука:
   `https://ваш-домен.com/api/payments/webhook`
3. Выберите события:
   - `checkout.session.completed`
   - `checkout.session.expired`
   - `payment_intent.payment_failed`

## 🎯 Тестовые карты Stripe

Для тестирования оплаты используйте:

**Visa:**
- Номер: `4242424242424242`
- Срок действия: 12/2034
- CVC: 123
- Результат: Успешная оплата

**Mastercard:**
- Номер: `5555555555554444`
- Срок действия: 12/2034
- CVC: 123
- Результат: Оплата с ошибкой (insufficient funds)

4000 0000 0000 0002 - карта будет отклонена
4000 0000 0000 9995 - недостаточно средств
4000 0000 0000 9987 - украденная карта

## 🚀 Запуск приложения

```bash
# Бэкенд
cd backend
npm run dev

# Фронтенд (в новом терминале)
cd ../frontend
npm start
```

## 📋 Проверка работы

1. Перейдите на страницу покупки абонемента
2. Выберите абонемент и нажмите "Оплатить"
3. В тестовом режиме используйте тестовые карты
4. Проверьте создание платежной сессии в бэкенде
5. Убедитесь, что вебхуки обрабатываются корректно

## 🔒 Важные замечания

- **Только тестовый режим** - реальные деньги списываться не будут
- **Логирование** - все платежные операции логируются в консоли
- **Безопасность** - никогда не храните секретные ключи в коде
- **Вебхуки** - убедитесь, что эндпоинт `/api/payments/webhook` доступен извне
