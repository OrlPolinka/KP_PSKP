## 🎯 Как запустить тесты

### 1. Unit тесты (работают всегда)

```bash
cd tests
npm run test:unit
```

**Результат:** ✅ 60/60 тестов пройдены

### 2. API тесты (требуют запущенный backend)

```bash
# Запустите backend сервер
cd backend
npm start

# В другом терминале запустите API тесты
cd tests
npm run test:api
```

### 3. Все тесты вместе

```bash
# Убедитесь что backend запущен
cd tests
npm test
```

### 4. Тесты с покрытием кода

```bash
npm run test:coverage
```

```bash

# 2. Запуск unit тестов (всегда работают)

# 3. Запуск backend (для API тестов)
cd ../backend && npm start

# 4. Запуск API тестов (в новом терминале)
cd ../tests && npm run test:api
```
